Git is een versiebeheersysteem (VCS)

Een VCS helpt je met het uit elkaar houden van verschillende versies van bestanden en mappen. Ook als je met meerdere ontwikkelaars aan hetzelfde project werkt.

Het is een log waarin staat wie, wat, wanneer heeft gedaan (en waarom door middel van commit messages).

  • Er zijn verschillende VCSen:
    • Non-distributed: Subversion, CVS
    • Distributed: Git, Mercurial
